

### EQUINOCCIO - DESARROLLOS
- Version de Node v18.16.0 (NPM v9.5.1)
- Vite

## Instalación

```bash
$ yarn install
```

## Configurar variables de entorno
- Crear archivo .env en funcion del template .env.template

## Correr la aplicación

```bash
# Desarrollo
$ yarn dev

# Build de produccion
$ yarn build
```
